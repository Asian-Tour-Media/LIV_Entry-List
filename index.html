How to sync data.json to GitHub

After exporting data.json, upload it to your GitHub repository alongside index.html:

  1. Go to your github.com/yourusername/liv-dashboard repository
  2. Click Add file → Upload files
  3. Drop in data.json (it will replace the old one automatically)
  4. Click Commit changes
  5. Wait ~30 seconds, then everyone who refreshes will get the latest data.
✓ The dashboard auto-fetches data.json from GitHub every time the page loads. If the file doesn't exist yet, it falls back to localStorage — so existing data is never lost.
